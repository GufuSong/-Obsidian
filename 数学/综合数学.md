
# 四元数基础

## Ⅰ. 导入

- 假设在一维坐标系上有一个常数 State ,  则这个常数的系数可以改变他的状态 .  例如 :  2 x State ,  则 State 伸长两倍 .  0.5 x State 则为缩短一半 .  -1 x State 则为向反方向延伸 ,  也可以理解为旋转 180 度 .  
- 而 1 作为系数时 ,  可以理解为什么都不做 .  数字作为状态 ,  被理解为变形后的 1 .  例如-3.5 表示拉长和反转后的 1 .
- 而 n 次方可以理解为 :  做了 n 次 .  例如 2 的 2 次方则代表做了两次拉伸 .  0 次方表示做了一次 , 也就是什么都不做 .  -1 次方表示对应操作的逆过程 ,  可以把状态复原 .

## Ⅱ. 四元数的三个轴

- 四元数是一个四维矢量 ,  `x,y,z`轴代表物体环绕旋转的轴向 ,  z代表旋转的角度 .

## Ⅲ. 四元数的变换方式

# AABB 包围盒

## Ⅰ. 射线相交轴对齐包围盒

>包围盒默认为轴对齐放行

```c++
float CheckAABB(float k, float3 Ro, float3 Rd, float3 SpherePos, float Radial)
{
    float3 t1 = (SpherePos - Radial * (1 + k * 0.1) - Ro) / Rd;
    float3 t2 = (SpherePos + Radial * (1 + k * 0.1) - Ro) / Rd;
    float3 tmin = min(t1, t2);
    float3 tmax = max(t1, t2);

    float tenter = max(tmin.x, max(tmin.y, tmin.z));
    float texit = min(tmax.x, min(tmax.y, tmax.z));

    if (tenter >= texit || texit <= 0)
    {
        return 0;
    }
    return 1;
}
```

**一. 参数列表解析 :**

- `k` 微调值 ,  可以将原来的包围盒尺寸放大或缩小 .
- `Ro` 射线的起点坐标 ,  用于计算射线与六个平面交代的参数值 .
- `Rd` 射线的方向矢量 .
- `SpherePos` 包围盒的中点
- `Radial` 半径 ,  用于顶底立方体边长的一半 .

**二. 代码逐行解读 :**

```c++
float3 t1 = (spherePos - Radial * (1 + k * 0.1) - Ro) / Rd;
```
- 作用 :  对应射线与每个维度盒子“最靠近射线起点”的那一侧平面交点的 **参数值** .
- `spherePos + -1.f * Radial * (1 + k * 0.1)` 包围盒中心点向靠近射线起点的位置移动(Radial , Radial , Radial) 矢量的距离 ,  近似的求出靠近射线包围盒面上的一个点
- `/ Rd` 已知 `Ro` 是射线的起点 ,  `Rd`是射线的方向 .  最后的交点为`P = spherePos + -1.f * Radial * (1 + k * 0.1)` 则推出以下公式 :
$$
P = Ro + t \cdot Rd
$$
- P , Ro , Rd 是已知的 ,  则t 等于
$$
t = \frac{P-Ro}{Rd}
$$
- t 是三维矢量 ,  代表 Rd 至 P 的系数 .

## 二. slab AABB包围盒

```c++
// https://iquilezles.org/articles/boxfunctions/

bool SlabAABB(folat3 inRayDirection, float3 inRayOrigin, float3 inPosition, float inBoxExtents)
{
	float3 m = -1.0 / inRayDirection;  //
	float3 n = m * (inRayOrigin - inPosition);
	float3 k = abs(m) * inBoxExtents;
	float3 t1 = -n - k;
	float3 t2 = -n + k;

	float tN = max(max(t1.x, t1.y), t1.z);
	float tF = min(min(t2.x, t2.y), t2.z);

	if( tN>tF || tF<1e-6)
	{
	    return false;
	}
	else if (tN > 1e-6)
	{
	    outHitTime = tN;
	    outHitPosition = inRayOrigin - (inRayDirection * tN);
	    outHitNormal = sign(inRayDirection) * step(t1.yzx,t1.xyz) *       step(t1.zxy,t1.xyz);
    
	    return true;
	}

	return false;
}
```

**一. 函数行为 :**
- 输入 `inRayDirection` 射线方向 , `inRayOrigin` 射线起点 , `inPosition` 碰撞盒位置 , `inBoxExtents` 碰撞盒范围 , 判断射线是否会穿过碰撞盒 .
